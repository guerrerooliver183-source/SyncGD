# Support

Need help with SyncGD? Here's how to get support:

## Common Issues

**Save not syncing across devices?**
- Make sure you're logged into the same Geometry Dash account on all devices
- Check that "Enabled" and "Auto-Load on Startup" are turned on in settings
- Verify you have a stable internet connection
- Try manually syncing via the Account menu

**Save retry not working?**
- Ensure "Save Retry" is enabled in settings
- Check that your save data isn't too large (profile + levels > 32MB may fail)
- Increase "Max Retry Attempts" or set to 0 for unlimited

**Notifications not showing?**
- Make sure "Show Notifications" is enabled
- Verify that `miskaa.notif` mod is installed and enabled
- Check your device's notification permissions

## Reporting Bugs

Please report bugs on the GitHub Issues page:  
https://github.com/guerrerooliver183-source/SyncGD/issues

Include:
- Geometry Dash version
- Platform (Windows/macOS/Android/iOS)
- Geode version
- SyncGD version
- Steps to reproduce the issue
